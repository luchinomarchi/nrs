Implementação Next.js

O projeto Next.js está estruturado em src/app e pronto para execução. Para rodar localmente, configure .env, execute npm install e depois npm run dev.

  • Configurações: tsconfig.json, next.config.js, tailwind.config.js, postcss.config.js
  • Estilos globais: src/app/globals.css
  • Layout raiz: src/app/layout.tsx
  • Autenticação mínima: src/app/api/auth/[...nextauth]/route.ts
  • Banco de dados: prisma/schema.prisma e .env.example
